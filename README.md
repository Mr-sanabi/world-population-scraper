# World Population Scraper

## Description

This script extracts country population data from a website and saves it into a CSV file.

## Features

* Parses HTML tables using BeautifulSoup
* Extracts structured data (country, population, changes, density, etc.)
* Saves data into CSV

## Technologies

* Python
* requests
* BeautifulSoup

## How to run

1. Install dependencies
2. Run the script
3. CSV file will be generated

## Output

CSV file with world population data by country
